# eBay Listings and Catalog Guide

> Complete reference for creating optimized listings, catalog matching, item specifics, SEO, multi-quantity management, and listing lifecycle.

---

## Listing Formats

### Fixed Price (Buy It Now)

The most common listing format. The item is listed at a set price until it sells or you end the listing.

| Feature | Details |
|---|---|
| **Duration** | Good 'Til Cancelled (GTC) — auto-renews every 30 days |
| **Best Offer** | Can be enabled to allow buyer negotiations |
| **Multi-quantity** | Can list multiple identical items in one listing |
| **Immediate Payment** | Can require buyer to pay immediately |
| **Promoted Listings** | Eligible for all ad types |

### Auction-Style

Items are listed for a set duration and buyers bid. Highest bidder wins.

| Feature | Details |
|---|---|
| **Duration options** | 1, 3, 5, 7, or 10 days |
| **Starting price** | You set the opening bid amount |
| **Reserve price** | Optional minimum price (additional fee: $5 or 7.5% of reserve, whichever is greater, partially refunded if item sells) |
| **Buy It Now** | Can add a BIN price to auctions (disappears once a bid is placed) |
| **Best for** | Rare/collectible items, items where market value is uncertain |
| **Immediate Payment** | Only available if Buy It Now is used |

### When to Use Each Format

| Situation | Recommended Format |
|---|---|
| Known market value, consistent demand | Fixed Price |
| Rare or unique item, uncertain value | Auction |
| Commodity items, multiple units | Fixed Price (multi-quantity) |
| Testing price point | Auction (to discover market value) |
| Time-sensitive inventory | Auction (short duration) |

## Creating a Listing: Detailed Process

### Navigation
```
Seller Hub > Listings > Create Listing
```

### Step 1: Product Identification

When you begin creating a listing, eBay asks you to identify your product. You have two paths:

**Path A: Catalog Match (Recommended for new/branded items)**
- Enter UPC, EAN, ISBN, or model number
- eBay matches to its product catalog
- Pre-fills title, item specifics, stock photo, category
- You add your own photos, price, condition, and shipping

**Path B: Standalone Listing (Required for unique/vintage/unbranded items)**
- Type a description of your item
- Select a category manually
- Fill in all item specifics yourself
- Provide all photos and details

| Catalog Match | Standalone Listing |
|---|---|
| Better search visibility for new products | Required for one-of-a-kind items |
| Pre-filled item specifics save time | Full control over all listing details |
| Buyers can compare across sellers | No comparison shopping widget |
| Required for some categories (e.g., media) | More work to create but more flexibility |

### Step 2: Category Selection

Choose the most specific subcategory possible. Correct categorization affects:
- Which item specifics are available/required
- Which final value fee rate applies
- Search result placement
- Which buyers find your listing

> **Tip**: If unsure, search for similar items on eBay and see which category they're listed in. You can also list in two categories for an additional insertion fee (equal to one extra insertion fee).

### Step 3: Condition

| Condition | Description | Use When |
|---|---|---|
| **New** | Brand new, unused, unopened, undamaged, in original packaging | Factory sealed |
| **New (Other)** | New but may be missing original packaging or have minor shelf wear | Unboxed new items |
| **Open Box** | Opened but never used, all accessories included | Customer returns, display models |
| **Certified Refurbished** | Professionally refurbished by manufacturer or qualified seller (requires eBay approval) | Refurbished electronics |
| **Seller Refurbished** | Restored by seller, fully functional | Items you've repaired |
| **Used** | Previously owned and used | Most secondhand items |
| **For Parts or Not Working** | Does not function as intended | Broken items for parts/repair |

> **Condition Description**: A text field (up to 1,000 characters) that appears prominently. For used items, always describe wear, scratches, missing parts, functionality issues. This is your primary defense against INAD returns.

## Item Specifics

Item specifics are structured data fields that describe your item's attributes. They are critical for search visibility.

### Required vs. Recommended

| Type | Impact | Examples |
|---|---|---|
| **Required** | Listing cannot be published without these | Brand, Type, Size (varies by category) |
| **Recommended** | Strongly advised — affects search ranking | Color, Material, Model, Features |
| **Custom** | You create the field name and value | Any additional attribute |

### Why Item Specifics Matter

1. **Search filters**: Buyers use left-sidebar filters to narrow results. If your item specifics are missing, your listing is excluded when a filter is applied
2. **Cassini ranking**: Listings with more complete item specifics rank higher
3. **Catalog matching**: Accurate specifics help eBay match your listing to the right product page
4. **Structured data**: Powers eBay's product-based shopping experience

### Best Practices for Item Specifics

- Fill in **every** available field, even optional ones
- Use eBay's suggested values when possible (standardized values rank better than custom text)
- Do not stuff keywords into item specifics — use accurate values
- For multi-packs, specify the unit quantity in the "Number of Items" or "Bundle Listing" specific
- Update item specifics when eBay adds new fields to your category

## Title Optimization

Your title is the single most important factor for search visibility. You have 80 characters — use them all.

### Title Formula

```
[Brand] [Model/Product Name] [Key Attribute 1] [Key Attribute 2] [Size/Capacity] [Condition Keyword]
```

### Title Best Practices

| Do | Don't |
|---|---|
| Use all 80 characters | Leave characters unused |
| Include brand name first | Start with generic words like "New" or "Great" |
| Add model number if applicable | Use ALL CAPS (except brand acronyms) |
| Include size, color, material | Use special characters (* ! @ # $ %) |
| Use words buyers actually search for | Include filler words (WOW, L@@K, AMAZING) |
| Spell out numbers when appropriate | Repeat the same word multiple times |
| Include relevant keywords naturally | Stuff unrelated keywords (spam) |

### Title Examples

**Bad**: `AMAZING!! Brand New Widget L@@K *** FREE SHIPPING ***`

**Good**: `Dyson V15 Detect Cordless Vacuum Cleaner Gold/Nickel Complete Kit New Sealed`

**Bad**: `Used Shoes Size 10`

**Good**: `Nike Air Max 90 Men's Running Shoes Size 10 White Black 2024 CW7483-100 Used`

## Photo Optimization

Photos are the second most important listing element after the title.

### Requirements and Limits

| Specification | Requirement |
|---|---|
| **Minimum photos** | 1 |
| **Recommended** | 8-12 |
| **Maximum** | 24 (free) |
| **Minimum resolution** | 500 x 500 pixels |
| **Recommended resolution** | 1600 x 1600 pixels (enables zoom) |
| **File formats** | JPEG, PNG, TIFF, BMP, GIF |
| **Max file size** | 12 MB per image |
| **Background** | White or light background preferred for first photo |

### Photo Checklist

1. **Main photo**: Clean, white background, product fills 80%+ of frame
2. **Multiple angles**: Front, back, sides, top, bottom
3. **Close-ups**: Brand labels, model numbers, serial numbers
4. **Flaws**: Every scratch, dent, stain, or imperfection (for used items)
5. **Scale reference**: Show item next to common object or include measurements
6. **Accessories**: Everything included in the sale
7. **Packaging**: If original packaging is included, show it
8. **Working condition**: Screen on for electronics, moving parts demonstrated

### What NOT to Include in Photos

- Watermarks or logos
- Text overlays or borders
- Collages (multiple images in one frame)
- Stock photos for used items (always photograph your actual item)
- Photos of items not included in the sale
- Promotional text or pricing in photos

## Description Best Practices

The item description appears below the main listing details. While less critical for search than titles and item specifics, a good description converts browsers to buyers and prevents returns.

### Description Structure

1. **Opening statement**: What the item is, briefly
2. **Condition details**: Honest assessment of wear/damage (for used)
3. **What's included**: List all items, accessories, manuals
4. **What's NOT included**: Explicitly state if stands, cables, etc. are missing
5. **Dimensions/measurements**: Actual measurements, not just size labels
6. **Shipping information**: Any special notes about shipping
7. **Return policy highlight**: Remind buyers of your return policy

### Description Tips

- Use plain HTML or eBay's built-in editor — avoid complex HTML/CSS
- Mobile-friendly: Most buyers browse on phones. Avoid wide tables, tiny fonts
- Do not include contact information (email, phone, website)
- Do not link to external websites
- Do not include calls-to-action to buy outside eBay
- Keep it scannable: Use bullet points and short paragraphs

## Cassini Search Algorithm: How eBay Ranks Listings

Cassini is eBay's search engine. Understanding its ranking factors helps you optimize listings for visibility.

### Primary Ranking Factors

| Factor | Weight | How to Optimize |
|---|---|---|
| **Relevance** | Highest | Accurate title, complete item specifics, correct category |
| **Listing quality** | High | High-resolution photos, complete description, competitive pricing |
| **Seller performance** | High | Top Rated status, high feedback score, low defect rate |
| **Best Match signals** | High | Click-through rate, conversion rate, watch count |
| **Free shipping** | Medium-High | Offer free shipping (build into item price) |
| **Free returns** | Medium | Offer 30-day or 60-day free returns |
| **Handling time** | Medium | Same-day or 1-day handling |
| **Item specifics completeness** | Medium | Fill in all available fields |
| **Recent sales velocity** | Medium | Listings with recent sales rank higher |
| **Price competitiveness** | Medium | Priced near or below comparable listings |

### Things That Hurt Your Ranking

| Factor | Impact |
|---|---|
| Below Standard seller level | Significant suppression |
| Duplicate listings for same item | Both listings penalized |
| Keyword stuffing in title | May be flagged as spam |
| High return rate | Reduced visibility |
| Frequently cancelled transactions | Listing suppression |
| Stock photos for used items | Lower conversion = lower ranking over time |
| Incomplete item specifics | Filtered out when buyers use search refinements |

## Multi-Quantity Listings

For sellers with multiple identical items, multi-quantity listings consolidate everything into one listing.

### How Multi-Quantity Works

- Set "Quantity" to the number of identical items you have
- Each sale deducts from the available quantity
- Sales history, watchers, and Best Match ranking accumulate on one listing
- Much better for search ranking than creating separate listings

### Variations (Multi-SKU Listings)

For items that come in different sizes, colors, or configurations:

```
Seller Hub > Create Listing > Add Variations
```

| Variation Feature | Example |
|---|---|
| Size | S, M, L, XL |
| Color | Black, White, Red |
| Style | V-Neck, Crew Neck |
| Capacity | 64GB, 128GB, 256GB |

Each variation can have its own:
- Price
- Quantity
- SKU
- UPC/EAN
- Photos

> **Benefits of variations**: Single listing accumulates all sales history, buyer can select options from one page, fewer listings to manage, better search ranking.

## Out-of-Stock Handling

When you run out of inventory, you have options to preserve your listing's search ranking and sales history.

### Out-of-Stock Feature

```
Seller Hub > Account > Site Preferences > Out-of-Stock Option > Enable
```

When enabled:
- Listings with 0 quantity are **hidden** from search (not ended)
- Sales history and watchers are preserved
- When you add quantity back, the listing immediately reappears
- No new insertion fee when restocking

| Action | Sales History Preserved | Insertion Fee on Restock |
|---|---|---|
| End listing manually | Lost (if not relisted within 30 days) | Yes (new listing) |
| Out-of-stock (0 quantity) | Preserved | No |
| Relist ended listing | Partially preserved | Yes |

> **Always enable out-of-stock handling.** It is one of the most important settings for any seller with recurring inventory.

## Revising and Ending Listings

### What You Can Revise

| Element | Can Revise? | Notes |
|---|---|---|
| Title | Yes | May affect search ranking temporarily |
| Price | Yes | Can increase or decrease |
| Photos | Yes | Can add, remove, or replace |
| Description | Yes | Append-only in some cases if bids exist |
| Item specifics | Yes | |
| Quantity | Yes | Can increase; decrease only to current sales level |
| Shipping | Yes | Cannot change after a sale on that listing |
| Category | Limited | Cannot change if listing has bids or sales in last 12 hours |
| Duration/Format | No | Cannot change auction to fixed price or vice versa |

### Revising Listings with Active Bids (Auction)

If an auction has bids:
- Cannot lower starting price
- Cannot add Best Offer
- Cannot reduce quantity below bid count
- Can add to description (appended, not replaced)
- Can add photos but not remove existing ones

### Ending a Listing

```
Seller Hub > Listings > Active > [Select listing] > End Listing
```

Reasons for ending:
- Item sold elsewhere
- Item damaged or lost
- Listing error (will relist correctly)
- No longer want to sell

> **Warning**: Ending and relisting loses sales history if not done within 30 days. Use the out-of-stock feature instead when possible.

## Listing Templates and Bulk Tools

### Business Policies (Shipping, Return, Payment Profiles)

Create reusable profiles so you don't configure shipping and returns for every listing:

```
Seller Hub > Listings > Business Policies
```

- **Shipping profiles**: Flat rate, calculated, free shipping — saved and reusable
- **Return profiles**: 30-day, 60-day, no returns — saved and reusable
- **Payment profiles**: Immediate payment required or not

### Bulk Listing Tools

| Tool | Best For | Access |
|---|---|---|
| **Seller Hub bulk editor** | Changing price, quantity, or item specifics across many listings | Seller Hub > Listings > Active > Bulk Edit |
| **File Exchange (CSV upload)** | Creating or revising hundreds/thousands of listings via spreadsheet | Seller Hub > Listings > Upload via CSV |
| **eBay Seller app** | Quick listing creation with phone camera | iOS/Android app |
| **Third-party tools** | High-volume sellers needing advanced features | inkFrog, SixBit, Sellbrite, ChannelAdvisor, etc. |

### Listing Templates (Seller Hub)

Save listing configurations as templates for repeated use:

1. Create a listing with your preferred settings
2. Before publishing, click "Save as Template"
3. Name the template
4. When creating new listings, select "Use a Template" to pre-fill settings

## Promoted Listings Integration

When creating or revising a listing, you can enable Promoted Listings:

| Promotion Type | How It Works | Fee |
|---|---|---|
| **Promoted Listings Standard** | Cost-per-sale. You set an ad rate %. Only charged when item sells via ad click | 2%-30%+ of sale price |
| **Promoted Listings Advanced** | Cost-per-click. You set a daily budget and bid per click | Per-click charge regardless of sale |
| **Promoted Listings Express** | Single-day promotions for auction listings | Fixed fee based on listing price |

> See the Promoted Listings reference for detailed strategy guidance.

## Common Listing Mistakes and Fixes

| Mistake | Risk | Fix |
|---|---|---|
| Missing item specifics | Filtered out of search | Fill in all fields, even optional ones |
| Stock photo for used item | INAD return, negative feedback | Always photograph actual item |
| Vague condition description | Buyer disputes, defects | Be specific: "2-inch scratch on back panel" |
| Title keyword stuffing | Search penalty | Use natural, accurate keywords |
| Wrong category | Wrong fees, wrong audience | Check competitor listings for correct category |
| Not enabling out-of-stock | Lost sales history when restocking | Enable in Account > Site Preferences |
| Duplicate listings | Both suppressed in search | Use multi-quantity or variation listings |
| Complex HTML description | Breaks on mobile | Use simple formatting, test on phone |
