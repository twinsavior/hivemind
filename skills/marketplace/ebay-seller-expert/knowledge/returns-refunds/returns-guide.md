# eBay Returns and Refunds Guide

> Complete reference for eBay Money Back Guarantee, return policies, INAD disputes, seller protections, partial refunds, escalation process, and appeals.

---

## eBay Money Back Guarantee

The eBay Money Back Guarantee (MBG) is eBay's buyer protection program. It covers virtually every purchase on the platform.

### What It Covers

| Coverage | Details |
|---|---|
| Item Not Received (INR) | Buyer did not receive the item |
| Item Not As Described (INAD) | Item received is materially different from listing description |
| Damaged in shipping | Item arrived damaged |
| Missing parts or accessories | Item is incomplete compared to listing |
| Wrong item received | Buyer received a different item than ordered |

### What It Does NOT Cover

- Buyer's remorse (unless seller offers returns)
- Items accurately described that buyer simply doesn't want
- Items that were tested/used after delivery (unless defective)
- Vehicles, real estate, classified ads, some business/industrial equipment
- Items picked up locally

### Timeline for Buyer Claims

| Step | Timeframe |
|---|---|
| Buyer can open a return/case | Within 30 days of actual or estimated delivery date |
| For INAD claims | Up to 30 days after delivery (or longer if seller offers 60-day returns) |
| Seller response window | 3 business days to respond to a return request |
| If seller doesn't respond | eBay may automatically close in buyer's favor |
| Escalation to eBay | Buyer can escalate after seller's response window expires |

## Return Policy Options

When creating listings, you choose a return policy. Your choice significantly affects search visibility and buyer confidence.

### Available Return Policies

| Policy | Details | Search Impact | TRS Plus Eligible? |
|---|---|---|---|
| **60-day free returns** | Buyer has 60 days; seller pays return shipping | Highest visibility | Yes |
| **30-day free returns** | Buyer has 30 days; seller pays return shipping | High visibility | Yes |
| **30-day buyer-paid returns** | Buyer has 30 days; buyer pays return shipping | Moderate visibility | No |
| **60-day buyer-paid returns** | Buyer has 60 days; buyer pays return shipping | Moderate visibility | No |
| **No returns accepted** | Only MBG claims (INAD, INR) are accepted | Lowest visibility | No |

> **Important**: Even with "no returns accepted," you MUST accept returns for INAD claims under the eBay Money Back Guarantee. "No returns" only applies to buyer's remorse/change of mind returns.

### Recommended Return Policy

For most sellers, **30-day free returns** is the optimal choice:
- Required for Top Rated Plus badge and 10% FVF discount
- Significant search visibility boost
- Increases buyer confidence and conversion rate
- The 10% FVF discount often offsets the cost of return shipping

## Buyer Return Reasons

When a buyer initiates a return, they select a reason. The reason determines who pays for return shipping and whether the return counts as a defect.

### Return Reasons and Their Impact

| Return Reason | Category | Who Pays Return Shipping | Counts as Defect? | Seller Must Accept? |
|---|---|---|---|---|
| **Doesn't fit** | Buyer's remorse | Depends on return policy | No | Only if returns accepted |
| **Changed my mind** | Buyer's remorse | Depends on return policy | No | Only if returns accepted |
| **Found a better price** | Buyer's remorse | Depends on return policy | No | Only if returns accepted |
| **Just didn't like it** | Buyer's remorse | Depends on return policy | No | Only if returns accepted |
| **Ordered by mistake** | Buyer's remorse | Depends on return policy | No | Only if returns accepted |
| **Item not as described** | INAD | Seller pays | Yes (if case closed against you) | Yes (MBG) |
| **Doesn't work or defective** | INAD | Seller pays | Yes (if case closed against you) | Yes (MBG) |
| **Missing parts or accessories** | INAD | Seller pays | Yes (if case closed against you) | Yes (MBG) |
| **Wrong item sent** | INAD | Seller pays | Yes (if case closed against you) | Yes (MBG) |
| **Arrived damaged** | INAD/Shipping | Seller pays | Yes (if case closed against you) | Yes (MBG) |

### Buyer's Remorse vs. INAD: Key Distinctions

| Aspect | Buyer's Remorse | INAD |
|---|---|---|
| Seller must accept? | Only if return policy allows | Always (Money Back Guarantee) |
| Return shipping | Per your return policy | Seller always pays |
| Restocking fee allowed? | Yes (if enabled in policy) | No |
| Defect impact | None | Potential defect if not resolved |
| Partial refund allowed? | No (full refund required for remorse returns) | Yes (with buyer agreement) |

## The Return Process: Step by Step

### When a Buyer Opens a Return

1. **Buyer initiates return**: Selects reason and provides details through eBay
2. **You receive notification**: Email + Seller Hub notification
3. **You have 3 business days to respond** with one of these options:

### Your Response Options

| Option | When to Use | What Happens |
|---|---|---|
| **Accept the return** | You agree with the return request | eBay generates a return label; buyer ships back |
| **Issue a full refund (keep the item)** | Item is low-value or not worth return shipping cost | Buyer keeps item, gets full refund, case closed |
| **Offer a partial refund** | Minor issue that doesn't warrant full return | Buyer can accept or decline; if declined, can escalate |
| **Send a message to buyer** | Need more information or want to negotiate | Does NOT count as an official response — you must still formally respond |
| **Decline the return** | Buyer's remorse return when you don't accept returns | Only valid for remorse reasons, NOT for INAD |

> **Warning**: If you do not respond within 3 business days, eBay may automatically close the case in the buyer's favor and issue a full refund from your funds.

### After You Accept the Return

1. eBay generates a prepaid return shipping label (charged to you for INAD; per your policy for remorse)
2. Buyer has 5 business days to ship the item back (extendable)
3. Tracking updates when buyer ships
4. When you receive the item back:
   - Inspect immediately
   - Issue refund within 2 business days of receiving the return
5. If buyer doesn't ship back within the window, you can close the return

### Refund Timing

| Scenario | Refund Deadline |
|---|---|
| You accept return and receive item back | Within 2 business days of receiving returned item |
| You offer refund without return | Immediately upon selecting this option |
| eBay closes case in buyer's favor | Automatic — eBay deducts from your account |

## Restocking Fees

For **buyer's remorse returns only** (not INAD), you can charge a restocking fee if your return policy specifies one.

### Restocking Fee Rules

| Condition | Maximum Restocking Fee |
|---|---|
| Item returned in original condition | No restocking fee recommended (hurts visibility) |
| Item returned used/damaged by buyer | Up to 50% restocking fee |
| Item returned missing parts | Up to 50% restocking fee |

### How to Enable Restocking Fees

```
Seller Hub > Listings > Business Policies > Returns > Edit > Restocking Fee
```

> **Note**: Restocking fees are NOT available on free returns listings. If you offer free returns, you cannot charge a restocking fee for remorse returns. However, you CAN deduct up to 50% for used/damaged returns even with free returns (see "partial refund for used returns" below).

### Deductions on Free Returns (Used/Damaged)

If you offer free returns and the buyer returns an item that has been used or damaged:

1. Accept the return and receive the item back
2. If the item is not in original condition, you may issue a partial refund (deduct up to 50%)
3. Document the condition with photos before and after
4. eBay may review if the buyer disputes your deduction

## Partial Refunds

### When You Can Offer a Partial Refund

| Scenario | Partial Refund Appropriate? |
|---|---|
| Minor flaw not mentioned in listing | Yes — INAD partial refund with buyer agreement |
| Missing accessory that was listed | Yes — proportional to missing item's value |
| Item works but has cosmetic damage from shipping | Yes — negotiate with buyer |
| Buyer wants to keep item but wants discount | Yes — if both parties agree |
| Buyer's remorse, full return policy | No — must issue full refund |

### How to Issue a Partial Refund

```
Seller Hub > Orders > Returns > [Select return] > Offer Partial Refund
```

1. Enter the partial refund amount
2. Add a message explaining why (be professional)
3. Buyer can accept or decline
4. If buyer declines, they can escalate to eBay
5. If buyer accepts, case is closed — no defect

## Item Not As Described (INAD): Deep Dive

INAD claims are the most impactful return type for sellers because they can result in defects.

### What Qualifies as INAD

| INAD Category | Examples |
|---|---|
| **Wrong item** | Sent size M when listing said size L; wrong color |
| **Defective/broken** | Electronics don't power on; mechanical parts broken |
| **Missing parts** | Listing said "complete set" but missing pieces |
| **Condition mismatch** | Listed as "New" but shows signs of use |
| **Authenticity issues** | Item is not genuine/authentic as described |
| **Material damage** | Significant undisclosed damage not shown in photos |

### What Does NOT Qualify as INAD (Even If Buyer Claims It)

| Situation | Why It's Not INAD |
|---|---|
| Buyer didn't read description carefully | Description was accurate |
| Item doesn't fit (clothing, parts) | Buyer chose wrong size, compatibility issue |
| Buyer found it cheaper elsewhere | Not a description issue |
| Item works but buyer expected higher quality | Subjective — description was accurate |
| Color looks different on buyer's screen | Photo showed accurate color |

### INAD Impact on Defect Rate

| Outcome | Defect? |
|---|---|
| You accept INAD return and refund before buyer escalates | No defect |
| You resolve with partial refund buyer accepts | No defect |
| Buyer escalates to eBay and eBay rules in buyer's favor | **YES — defect** |
| Buyer escalates and eBay rules in your favor | No defect |
| You don't respond within 3 business days | **YES — defect** (automatic close in buyer's favor) |

> **Key takeaway**: Always respond to INAD claims within 3 business days. Even if you disagree, respond. An unanswered case is an automatic defect.

### INAD Prevention Strategies

1. **Photograph everything**: All angles, all flaws, all accessories
2. **Write detailed condition descriptions**: Mention every scratch, stain, missing part
3. **Test electronics before listing**: Confirm functionality, battery health, etc.
4. **Use accurate item specifics**: Correct brand, model, size, color, material
5. **Ship securely**: Damage in transit becomes your INAD problem
6. **Keep proof**: Photos of item before shipping, packaging process, receipt of purchase

## When eBay Automatically Sides with the Buyer

eBay has automated resolution systems that close cases in the buyer's favor without human review in certain situations:

| Situation | eBay Action |
|---|---|
| Seller doesn't respond within 3 business days | Auto-refund buyer, defect applied |
| Tracking shows item not delivered and seller has no proof | Auto-refund buyer |
| No tracking uploaded at all | Auto-refund buyer |
| Signature confirmation missing on $750+ item, buyer claims INR | Auto-refund buyer |
| Seller's return address is invalid and return cannot be delivered | Auto-refund buyer |
| Seller doesn't refund within 2 business days of receiving return | eBay may auto-refund |

## Seller Protections

eBay does offer seller protections in specific scenarios.

### When eBay Protects the Seller

| Protection | Requirements |
|---|---|
| **Item Not Received claim — tracking shows delivered** | Valid tracking showing delivery to buyer's address (or confirmed nearby) |
| **High-value INR — signature confirms delivery** | Signature confirmation for $750+ items delivered to buyer's address |
| **Buyer remorse when no returns offered** | Seller can decline remorse-based returns if "no returns" policy |
| **Buyer returns wrong/different item** | Document what you received; report to eBay with photos |
| **Feedback extortion** | Buyer threatens negative feedback to extract refund — report to eBay |
| **Policy-protected transactions** | Items excluded from MBG (vehicles, real estate, some categories) |

### Documenting for Seller Protection

- **Before shipping**: Photograph item, serial number, packaging
- **At shipping**: Use tracking (signature for $750+), photograph package before carrier handoff
- **At return receipt**: Photograph package condition, open and photograph contents, document any discrepancies

## Appeal Process for Unfair Returns

If eBay closes a case against you and you believe it was incorrect, you can appeal.

### How to File an Appeal

```
Seller Hub > Performance > Service Metrics > [Click on the metric] > View Details > [Find the transaction] > Appeal
```

Or contact eBay seller support directly:
- Phone: 1-866-540-3229
- Chat: Through Seller Hub > Help > Contact Us

### Valid Appeal Reasons

| Reason | Evidence Needed |
|---|---|
| Tracking confirms delivery but case was lost | Tracking number showing delivered status |
| Buyer admitted fault in eBay messages | Screenshot/link to the message thread |
| Item was accurately described, buyer has remorse | Listing photos and description proving accuracy |
| Buyer returned a different/empty item | Photos of what was returned, original item serial number, weight comparison |
| eBay system error | Documentation of the error |

### Appeal Timeline

- File within **30 days** of case closure
- eBay reviews within 5-10 business days (can be longer)
- You'll be notified by email of the outcome
- If appeal succeeds: defect removed, refund reversed (if applicable)
- If appeal fails: You can escalate further through eBay's executive customer relations

### Tips for Successful Appeals

1. Be factual, not emotional — present evidence clearly
2. Reference specific eBay policies that support your case
3. Include all relevant tracking, photos, and message transcripts
4. Be specific about what resolution you're requesting (defect removal, refund reversal, etc.)
5. If phone support denies your appeal, try again with a different representative — outcomes can vary

## Return Shipping Cost Considerations

### Who Pays for Return Shipping

| Scenario | Who Pays |
|---|---|
| INAD claim (any return policy) | Seller always pays |
| Buyer's remorse + free returns policy | Seller pays |
| Buyer's remorse + buyer-paid returns | Buyer pays |
| Buyer's remorse + no returns policy | Return not required (but INAD still applies) |

### Return Shipping Label Options

| Option | How It Works |
|---|---|
| **eBay-generated label** | eBay creates a prepaid label and charges seller; discounted rates |
| **Seller-provided label** | You email or message a return label to the buyer |
| **Buyer purchases their own label** | For buyer-paid returns; buyer ships and provides tracking |

> **Recommendation**: Use eBay-generated return labels when possible. They're discounted, tracking is automatic, and the process is seamless for both parties.

## Return Fraud and How to Handle It

### Common Return Fraud Types

| Fraud Type | Description | Response |
|---|---|---|
| **Empty box** | Buyer returns an empty or near-empty package | Document package weight at delivery vs. shipped weight; file appeal |
| **Switcheroo** | Buyer returns a different or damaged item | Compare serial numbers, photos; report to eBay |
| **Wear and return** | Buyer uses item then returns it claiming INAD | Document condition differences with photos; partial refund or appeal |
| **False INAD** | Item is exactly as described but buyer claims INAD for free return shipping | Provide evidence listing is accurate; appeal if case is lost |

### Protecting Against Return Fraud

1. **Record serial numbers** on electronics, luxury goods
2. **Photograph item weight on scale** before shipping
3. **Video record packing process** for high-value items
4. **Use tamper-evident seals or stickers** on high-value item packaging
5. **Compare return package weight** to what you shipped — significant difference is evidence
6. **Report fraud immediately** to eBay through the resolution center
