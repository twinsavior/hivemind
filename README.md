## Get Started

### Prerequisites (required for ALL install methods)

HIVEMIND uses Claude as its brain. You need [Node.js](https://nodejs.org/) 22+ and the Claude Code CLI:

```bash
npm install -g @anthropic-ai/claude-code
```

Then run `claude` once to authenticate with your Anthropic account.

> **Stuck on "Starting your swarm"?** This almost always means `claude` isn't installed or isn't on your PATH. Run `claude --version` in your terminal to check.

---

### Option A: Download the Desktop App (easiest)

1. **Mac**: [Download the .dmg](https://github.com/twinsavior/hivemind/releases/download/v1.0.0-beta.7/HIVEMIND-1.0.0-beta.7-arm64.dmg) вҶ’ open it вҶ’ drag HIVEMIND to Applications
   - First launch: right-click the app вҶ’ Open (macOS requires this for unsigned apps)
2. **Windows**: [Download the .exe](https://github.com/twinsavior/hivemind/releases/download/v1.0.0-beta.7/HIVEMIND-Setup-1.0.0-beta.7.exe) вҶ’ run it вҶ’ done
3. Launch HIVEMIND. The onboarding wizard walks you through everything.

### Option B: Run from Source

```bash
# 1. Clone and install
git clone https://github.com/twinsavior/hivemind.git
cd hivemind
pnpm install        # requires pnpm 10+ (npm install -g pnpm)

# 2. Build the server
pnpm build

# 3. Run the desktop app
cd desktop
npm install
npm start
```

> **Build tools needed:** macOS: `xcode-select --install` В· Linux: `apt install build-essential python3` В· Windows: install Visual Studio Build Tools

## What is HIVEMIND?

HIVEMIND is a desktop app that gives you an AI-powered team for your reselling business. It runs a swarm of 6 specialized agents вҖ” led by **Nova**, your AI co-founder вҖ” that handle the daily grind: parsing purchase emails, tracking profitability, monitoring account health, and managing your source-to-sale pipeline.

Everything runs locally on your machine. Your data never leaves your computer. No cloud, no subscription fees for the platform itself.

**Built for the Buy Box / FlipAlert seller community.**

---

## вңЁ Features

### Seller Operations
- **Email automation** вҖ” Parses purchase confirmations, shipment tracking, and seller alerts from 15+ retailers (Amazon, Walmart, eBay, Target, Costco, etc.) via Gmail OAuth or IMAP. Multi-account support with validated connections and encrypted credential storage
- **Marketplace APIs** вҖ” Connects to Amazon SP-API, Walmart Seller Center, and eBay REST APIs for orders, inventory, FBA shipments, and account health
- **Financial tracking** вҖ” SimpleFIN Bridge integration syncs bank/credit card transactions, auto-categorizes spend (sourcing, prep, shipping, fees)
- **Profitability calculator** вҖ” Per-item ROI across Amazon, Walmart, and eBay using exact fee lookups
- **Source-to-sale pipeline** вҖ” Tracks items through 8 stages: Purchased вҶ’ Shipped to Prep вҶ’ Prepping вҶ’ Shipped to FBA вҶ’ Received вҶ’ Live вҶ’ Sold вҶ’ Settled
- **Account health monitoring** вҖ” Alerts for suspensions, policy violations, IP complaints, and performance warnings

### AI Agent Swarm
- **5 specialized agents** + Nova coordinator with autonomous task delegation
- **Claude Code** вҖ” powered by Anthropic's Claude, the best coding & reasoning model
- **Trust-based security** вҖ” Owner, Trusted, and Untrusted tiers with per-task permissions
- **Hierarchical memory** вҖ” Persistent context across conversations (L0/L1/L2)
- **Skills system** вҖ” Extensible skills with hot-reload

### Platform
- **Desktop app** вҖ” Native Electron app with chat UI and operations dashboard
- **Discord integration** вҖ” Connect HIVEMIND to your Discord server for bidirectional messaging
- **Local-first** вҖ” All data stays on your machine in SQLite databases
- **Open source** вҖ” MIT licensed, community-driven

## вҡҷпёҸ Configuration

HIVEMIND uses a layered config model: **CLI args > environment variables > hivemind.yaml > defaults**.

### Marketplace Connections

Marketplace API credentials (Amazon SP-API, Walmart, eBay) are configured through the onboarding flow in the desktop app and stored encrypted locally.

### Email Setup

Email accounts (Gmail OAuth or IMAP) are configured through the desktop app's Settings > Email section.

### Discord

Connect HIVEMIND to your Discord server through Settings > Connectors. The setup wizard walks you through creating a bot and linking it.
