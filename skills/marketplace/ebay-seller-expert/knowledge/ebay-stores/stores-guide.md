# eBay Stores — Complete Seller Guide

## Overview

An eBay Store subscription provides reduced fees, free listings, marketing tools, and branding capabilities. There are five subscription tiers, each designed for different selling volumes and business needs.

---

## Store Subscription Tiers

### Pricing and Billing

| Tier | Monthly (month-to-month) | Monthly (annual plan) | Best For |
|---|---|---|---|
| Starter | $7.95/mo | $4.95/mo | Casual sellers with 50+ listings |
| Basic | $27.95/mo | $21.95/mo | Part-time sellers with 250+ listings |

## Setting Up Your Store

### Subscribing to a Store

**Navigation:** My eBay > Account > Subscriptions > eBay Store, or go to `ebay.com/str/signup`

1. **Choose your tier** based on your listing volume and needs.
2. **Select billing plan:** Monthly or annual.
3. **Enter payment information** (or confirm existing on-file payment method).
4. **Name your store:** Choose a name that reflects your brand (alphanumeric, up to 35 characters).
5. **Confirm subscription.**

### Initial Store Setup Checklist

| Task | Navigation | Priority |
|---|---|---|
| Set store name | Store > Store settings > Store name | High |
| Upload store logo | Store > Store settings > Logo (300x300px recommended) | High |
| Write store description | Store > Store settings > Description | High |
| Choose store theme | Store > Store settings > Display preferences | Medium |
| Create store categories | Store > Store categories > Add category | High |
| Set up store policies | Store > Store settings > Policies | Medium |
| Add store banner | Store > Store settings > Billboard image (1280x290px) | Medium |
| Configure vacation settings | Store > Store settings > Vacation mode | Low (until needed) |

## Store Categories

### Why Categories Matter

- Help buyers browse your store by product type.
- Improve the shopping experience and encourage multi-item purchases.
- Can be used as filters in Promoted Listings campaigns.
- Appear in your store's left navigation sidebar.

### Setting Up Categories

**Navigation:** Seller Hub > Store > Store categories (or My eBay > Store > Manage my Store > Store categories)

| Feature | Detail |
|---|---|
| Maximum categories | 300 |
| Category levels | Up to 3 levels deep (parent > child > grandchild) |
| Items per category | A listing can be in one store category |
| Naming | Keep names clear and buyer-friendly |
| Ordering | Drag and drop to reorder; most popular categories at top |

### Category Best Practices

1. Mirror eBay's main categories where possible for familiarity.
2. Use 5–15 top-level categories (not too many, not too few).
3. Create subcategories only if you have 10+ items in the parent category.
4. Update categories seasonally (add "Holiday Gifts" in Q4, etc.).
5. Move items to the correct store category when relisting.

### Branding Tips

1. **Logo:** Use a clean, professional logo. If you do not have one, use your store name in a clean font.
2. **Banner:** Show your best products or a seasonal message. Update quarterly.
3. **Description:** Include what you sell, your value proposition, and shipping/return highlights.
4. **Consistency:** Use the same logo, colors, and name across eBay, social media, and packaging.
5. **Professional appearance:** Avoid clip art, low-resolution images, or overly busy designs.

## Markdown Manager (Sale Events)

### What Markdown Manager Does

Markdown Manager lets you schedule temporary price reductions on your listings, with strikethrough pricing that shows the original price and the sale price.

**Navigation:** Seller Hub > Marketing > Markdown Manager

### Creating a Sale Event

1. **Name your event** (e.g., "Spring Clearance Sale").
2. **Set start and end dates** (minimum 1 day, maximum 60 days).
3. **Select items:** Choose specific listings or use category/store-category filters.
4. **Set markdown percentage:** 5%–80% off.
5. **Preview and launch.**

### Markdown Manager Details

| Feature | Detail |
|---|---|
| Minimum discount | 5% |
| Maximum discount | 80% |
| Minimum duration | 1 day |
| Maximum duration | 60 days |
| Maximum concurrent events | 25 |
| Items per event | Up to 500 listings per event |
| Stacking with Promotions Manager | Markdowns and promotions can apply to the same items |
| Display | Strikethrough pricing with "X% OFF" badge in search results |

### Markdown Manager Best Practices

1. **Schedule sales in advance** — create events ahead of time for holidays.
2. **Use compelling discount percentages** — 10% off is barely noticeable; 20%+ gets attention.
3. **Time your sales** around eBay sitewide events (eBay Holiday Sales, etc.).
4. **Combine with Promoted Listings** for maximum visibility during sales.
5. **Clear slow-moving inventory** with deeper markdowns (40%+).

## Promotions Manager

### Available Promotion Types

**Navigation:** Seller Hub > Marketing > Promotions > Create a promotion

| Promotion Type | What It Does | Example |
|---|---|---|
| Order discount | Percentage or flat discount based on minimum order value | "Spend $50, get 10% off" |
| Shipping discount | Free or reduced shipping for qualifying orders | "Free shipping on orders over $35" |
| Codeless coupon | Generates a shareable link; discount applies automatically when used | Share on social media for 15% off |
| Volume pricing | Discount for buying multiple of the same item | "Buy 2, get 5% off" |
| Buy One Get One | BOGO deal structure | "Buy 1 shirt, get 2nd 50% off" |

### Creating an Order Discount

1. **Set minimum purchase amount** (e.g., $50).
2. **Set discount type and amount** (percentage or dollar amount).
3. **Select eligible items** (all store items, specific categories, or specific listings).
4. **Set start and end dates.**
5. **Add a promotional message** (appears on listing pages).
6. **Launch.**

### Codeless Coupons

Codeless coupons are powerful for external marketing:

1. **Create the coupon** with your desired discount.
2. **Copy the shareable link.**
3. **Share on social media, email, or other channels.**
4. When a buyer clicks the link and purchases, the discount applies automatically — no code needed.
5. **Set a maximum redemption limit** to control costs.

| Setting | Options |
|---|---|
| Discount type | Percentage off or dollar amount off |
| Minimum purchase | Optional |
| Maximum redemption | Set a total cap on uses |
| Duration | Set start and end dates |
| Eligible items | All items, selected categories, or specific listings |

### Promotions Manager Best Practices

1. **Always have at least one active promotion** — it adds a "Special offer" banner to your listings.
2. **Stack promotions strategically** — order discount + free shipping gives buyers multiple incentives.
3. **Use codeless coupons for social media marketing** — trackable, shareable, no friction.
4. **Set reasonable minimum purchase amounts** — encourage larger cart sizes without being prohibitive.
5. **Run volume pricing on consumable or collectible items** — encourages multi-unit purchases.

## Vacation Mode

### What Vacation Mode Does

- Hides your active listings from search results and browse pages.
- Displays a vacation message on your store page.
- Prevents new sales during the vacation period.
- Does NOT end or remove your listings.
- Listings return to active status when you turn vacation mode off.

### Setting Up Vacation Mode

**Navigation:** Seller Hub > Store > Store settings > Vacation settings

| Setting | Option |
|---|---|
| Turn on vacation | Set start date (immediate or scheduled) |
| Display message | Custom message shown on your store page |
| Return date | Set expected return date (optional; displayed to buyers) |
| Auto-reply to messages | Enable with a custom message |
| Hide listings | Choose to hide from search or keep visible (not recommended during long absences) |

### Vacation Mode Tips

1. **Set up before you leave** — do not wait until the last minute.
2. **Process all pending orders** before activating vacation mode.
3. **Extend handling time** as an alternative for short breaks (1–3 days) instead of full vacation mode.
4. **Notify repeat buyers** about your planned absence.
5. **Keep vacation mode as short as possible** — extended absences hurt search ranking recovery.

### Impact on Search Ranking

- Listings lose search visibility during vacation mode.
- When you return, it may take 3–7 days for listings to regain their previous search positions.
- eBay's algorithm favors active, regularly updated stores.

## Choosing the Right Store Tier

### Decision Framework

| If You... | Recommended Tier |
|---|---|
| List 50–250 items, sell casually | Starter |
| List 250–1,000 items, sell regularly | Basic |
| List 1,000–10,000 items, sell full-time | Premium |
| List 10,000–25,000 items, high-volume business | Anchor |
| List 25,000+ items, enterprise-level operation | Enterprise |

### Break-Even Analysis

Calculate whether a Store subscription saves you money:

```
Monthly insertion fee savings = (Listings over free allowance x insertion fee without Store) - Store subscription cost

Monthly FVF savings = Total sales x (non-Store FVF rate - Store FVF rate)

Total monthly savings = Insertion fee savings + FVF savings - Store subscription cost
```

**Example: Is Basic worth it?**
- 500 active fixed-price listings.
- Without Store: 500 x $0.35 = $175/month in insertion fees.
- With Basic Store: 1,000 free listings, $0 insertion fees.
- Store cost: $21.95/month.
- **Net savings: $175 - $21.95 = $153.05/month.**

### Upgrading and Downgrading

- **Upgrading:** Takes effect immediately. Prorated for the remainder of the billing period.
- **Downgrading:** Takes effect at the end of the current billing period.
- **Cancelling:** On an annual plan, early cancellation incurs fees. On monthly, cancels at end of billing period.

**Navigation:** My eBay > Account > Subscriptions > Change or cancel Store subscription
