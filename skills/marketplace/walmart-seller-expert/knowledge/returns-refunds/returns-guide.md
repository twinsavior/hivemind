# Walmart Marketplace Returns & Refunds Guide

## Overview

Walmart Marketplace sellers must comply with the platform's return policy to maintain account standing. The returns process differs depending on whether orders are seller-fulfilled or fulfilled through Walmart Fulfillment Services (WFS). Sellers are expected to provide a seamless return experience that meets Walmart's customer satisfaction standards.

---

## Return Policy Overview

### Core Requirements
- All sellers must support a return policy that meets or exceeds Walmart's minimum requirements
- Customers must be able to initiate returns easily through Walmart.com or in Walmart stores
- Sellers cannot charge customers return shipping fees
- Restocking fees are generally prohibited (exceptions below)
- Offering discounts, alternative replacements, store credit, or incentives to avoid a return is not allowed

---

## Seller-Fulfilled Returns

### Standard Return Window
- **Minimum**: 30 days, beginning 7 days after the item's ship date
- Certain product categories have different windows:
  - **Electronics**: May have shorter return windows
  - **Major appliances**: May have shorter return windows
  - **Luxury items**: May have shorter return windows
- Customers who initiate returns within the window must receive refunds, even if the return finalization occurs after the window closes

### Return Centers
- Sellers must establish at least one return center with a valid U.S. address
- P.O. boxes are prohibited as return center addresses
- Excluded locations: Hawaii, Alaska, and U.S. territories
- The return center display name cannot include "Walmart" or any names associated with Walmart

### Return Shipping and Labels
- **Default carrier**: Walmart's Returns Shipping Service (RSS)
- Sellers may use FedEx or UPS instead by providing carrier account details
- Sellers cannot charge customers return shipping fees under any circumstances
- Sellers bear the cost of return shipping when the return is due to a product issue

### Seller-Responsible Return Reasons
When the return is due to a product issue, the seller pays for return shipping:
- Wrong size or poor fit
- Late to arrive
- Poor quality
- Missing parts or instructions
- Incorrect item received
- Not as described
- Item damaged
- Shipping box damaged
- Received extra item or quantity
- Received less than ordered
- Duplicate or defective item

---

## Refund Processing

### Timing Requirements
- **In-store returns**: Refunds issued immediately upon store receipt
- **Online returns (seller-fulfilled)**: Refunds issued when the carrier first scans the returned package
- **Deadline**: All refunds must be issued within 48 hours of the item being received at the return center
- **Auto-refund**: Walmart will automatically refund the customer if the seller does not process the refund within 48 hours

### Refund Methods
- Refunds are processed back to the customer's original payment method
- Walmart Customer Care may issue refunds on behalf of sellers for customer care issues
- Sellers cannot dispute auto-refunds triggered by failure to process within 48 hours

---

## WFS Returns (Walmart Fulfillment Services)

### WFS Return Window
- **Standard**: 90 days for most items (significantly longer than seller-fulfilled minimum of 30 days)
- Exceptions apply to wireless phones, major appliances, and other specific categories with shorter windows
- Extended holiday return windows may apply during seasonal periods

### WFS Return Methods
- Customers can print shipping labels for mail-in returns
- Customers can bring returns to any of nearly 5,000 Walmart stores across the U.S.

### WFS Return Processing
Items arriving at WFS return centers follow three paths:

1. **Sellable condition**: Item is restocked into inventory with no restocking fees
2. **Unsellable inventory**: Either returned to seller or disposed of, based on seller's customized return rules/preferences
3. **Walmart at fault returns** (damaged in transit, lost, wrong item shipped by WFS): Walmart retains the inventory and handles the return

### WFS Return Shipping Costs
- No shipping fees are charged to the customer
- Walmart covers the cost of return shipping for returns identified as Walmart at fault (damaged, late delivery, lost in transit, wrong item)
- Sellers pay for returns not deemed Walmart's responsibility

### WFS In-Store Returns
- Walmart associates receive items, verify eligibility, and route them to WFS return centers
- Status shows as "In Transit" without detailed tracking until processing completes
- In-store returns receive immediate refunds

### WFS Return Preferences
- Sellers can set preferences for handling unsellable returned inventory
- Options include having items returned to the seller or disposed of by Walmart
- Preferences are configured in WFS settings within Seller Center

---

## Keep It Rules

### Overview
Keep It Rules allow sellers to provide refunds while letting customers keep the item, which is often more cost-effective than processing returns with return shipping fees.

### Configuration Parameters
- **Product Department**: Which categories the rule applies to
- **Item price limit**: Maximum dollar amount per item for the rule to apply
- **Return Order limit**: Maximum total order value

### Partial Keep It Rules
- Additional parameters include Return Reason and percentage refund off
- Cannot apply to items with return exemptions
- Item-level rules require bulk template submission for SKUs outside specific departments or price limits (four-hour processing window)

### Important Notes
- If customers decline Keep It offers or transactions are ineligible, Walmart has sole discretion to handle the disposal of the item (sale, refurbishment, liquidation, donation, recycling, or destruction)
- In-store returns initiated without online requests bypass Keep It Rules entirely
- Sellers cannot negotiate alternative solutions outside pre-approved Partial Keep It Rules without explicit Walmart authorization

---

## Replacement Rules

- Sellers may offer replacements instead of refunds for certain return reasons
- Applicable for: damaged items, incorrect items received
- Must be configured in Seller Center

---

## Return Exemptions

### Exempt Item Categories
Certain items require exemption approval before listing:
- Custom or personalized items
- Luxury items
- Freight items (oversized/heavy)
- Hazardous materials (hazmat)

### How Exempt Returns Work
- Customers contact the seller directly to initiate returns for exempt items
- Up to 20% restocking fee may be charged for return exemption items (the only case where restocking fees are allowed)
- Sellers must apply for and receive approval for exemption status

### Non-Returnable Items
- Certain product types designated as "Final Sale" cannot be returned
- Items requiring assembly must be fully disassembled before return

---

## In-Store Returns

### Eligible Items
- Walmart accepts eligible Marketplace returns in-store on behalf of sellers
- This applies to both seller-fulfilled and WFS items

### Ineligible for In-Store Return
- Oversized or freight items
- Hazardous materials
- Luxury items
- Items with return exemptions

---

## Reimbursement and Disputes (WFS)

### WFS Reimbursements
- If Walmart damages or loses inventory while in their fulfillment centers, sellers can request reimbursement
- Reimbursement is based on the item's sale price minus applicable fees

### Dispute Process
- Sellers can dispute return decisions through Seller Center
- Documentation may be required to support the dispute
- Disputes are reviewed by Walmart's support team

---

## Customer Returns Reports

### Monitoring Returns
- Access return data through Seller Center reporting
- Track return reasons to identify product quality issues
- Monitor Return Rate metric (threshold: less than or equal to 6%, effective spring 2026)
- Use Item Not Received Rate metric to track delivery issues (threshold: less than or equal to 2%, effective spring 2026)

### Performance Impact
- High return rates can trigger account suppression or suspension
- Returns contribute to the Refund Rate metric (threshold: less than or equal to 6%)
- Pattern analysis helps identify problematic products or listings

---

## Prohibited Practices

1. Charging customers return shipping fees
2. Charging restocking fees (except for return exemption items, up to 20%)
3. Charging duty or import fees to customers
4. Offering discounts, replacements, store credit, or incentives to avoid a return
5. Haggling with customers about returns
6. Negotiating outside of pre-approved Keep It or Partial Keep It rules
7. Failing to process refunds within 48 hours of receipt

---

## Key Reminders

1. Seller-fulfilled minimum return window: 30 days (starting 7 days after ship date)
2. WFS return window: 90 days for most items
3. Refunds must be processed within 48 hours of receiving the returned item
4. No return shipping charges to customers under any circumstances
5. Keep your Return Rate below 6% and Refund Rate below 6%
6. Use Keep It Rules strategically to reduce return shipping costs
7. Monitor return reasons to improve product quality and listings
